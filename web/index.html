Project Details

设计说明

这是一个基于 Web 端的高级三维动态交互系统。模型通过精确的法线解耦算法实现了“部件炸开”视觉，支持在极近环境下的高保真特写以及全局自动旋转巡航。
交互指南：
• 首次点击：镜头从特写平滑拉远至中心，并开启鼠标拖拽旋转。
• 再次点击：模型部件瞬间炸开，1-2秒后自动恢复完整状态。
